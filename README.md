Deskripsi Singkat
SIBAJA adalah aplikasi web edukasi sederhana yang saya buat untuk membantu anak-anak (atau siapa saja) belajar bahasa Jawa dengan cara menyusun huruf menjadi kata yang benar.
Fokus utama: Sinau Boso Jowo secara interaktif dan menyenangkan.

Demo Online
https://dycan-mar.github.io/SIBAJA-SinauBahasaJawa-/

Teknologi yang Digunakan
-> HTML5, CSS, Vanilla JavaScript

Library yang Digunakan
-> Bootstrap5, jQuery, Google Font, SweetAlert, Loading Overlay jQuery, font Awesome  

Cara Menjalankan Lokal
git clone https://github.com/dycan-mar/SIBAJA-SinauBahasaJawa-.git
cd festika
Gunakan server lokal untuk menjalankan misal live server(extensions dari vscode) karena saya melakukan rooting sederhana menggunakan js
buka index.html pada server lokal
